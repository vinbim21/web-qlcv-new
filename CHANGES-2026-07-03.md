## 1. /admin/catalog — Bulk edit dự án: thêm "Đổi Hạng mục" + dropdown creatable

**Trước:** Thanh bulk chỉ có Đổi Dự án / Loại hình / Khối-Hệ thống / Bắt đầu / Đóng gói / Nhân bản / Xóa. Đổi Loại hình dùng `<Select>` id cố định; Hạng mục và Khối/Hệ thống là input tự do (không gợi ý dữ liệu có sẵn).

**Thay đổi:**
- Thêm action **"Đổi Hạng mục"** vào thanh bulk (field `name`).
- **Đổi Loại hình**: chuyển từ `<Select>` theo id sang `SearchableCombobox creatable` theo mã — gõ mã có sẵn để chọn, hoặc nhập mã mới để tự tạo `ConstructionType` (qua `upsertConstructionTypeReturnId`).
- **Đổi Hạng mục** / **Đổi Khối/Hệ thống**: input tự do → `SearchableCombobox creatable` với options lấy từ toàn bộ `name`/`blockSystem` hiện có trong `projects`.
- **Đổi Dự án**: thêm nút bút chì cạnh dropdown chọn dự án để **sửa tên/mã** ProjectGroup đang chọn ngay trong modal (gọi `saveProjectGroup`), không cần thoát ra tab "Quản lý dự án".

## 3. /manage — Gộp "Thêm tương tự" vào modal "Giao việc"

**Trước:** "Giao việc" (nút góc trên) và "Thêm tương tự" (thanh bulk khi chọn 1 việc) là 2 modal riêng, code trùng lặp.

**Thay đổi:**
- Gộp thành 1 state `assignModal: false | true | TaskRow` — `false` đóng, `true` mở trống, `TaskRow` mở kèm điền sẵn dòng 1 theo việc đó.
- Nút **"Giao việc"** giờ cũng thông minh: nếu đang chọn đúng 1 việc → tự điền sẵn (giống "Thêm tương tự"); không chọn gì → mở trống như cũ.
- Điền sẵn: tự chuyển đúng tab nhóm công việc của task, set Dự án/Loại hình/Hạng mục/Khối/Bộ môn/Công việc/Giai đoạn/Ưu tiên/ngày bắt đầu-kết thúc vào dòng 1 (tái dùng `AssignClient` với prop `defaultWorkGroupId` + `prefillRow` có sẵn).
