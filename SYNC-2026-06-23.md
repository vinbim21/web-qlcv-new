## Bước 4 — Build và khởi động lại server

```bash
# Kill server đang chạy (nếu có)
# Windows:
Get-NetTCPConnection -LocalPort 3000 -State Listen -ErrorAction SilentlyContinue |
  Select-Object -ExpandProperty OwningProcess |
  ForEach-Object { taskkill /PID $_ /F }

# Build
npm run build

# Start
npm start
```

## Lưu ý

- Nếu deploy lên **Vercel**: push nhánh `vunt38` lên GitHub, Vercel tự build.  
  Vercel **không cần** chạy `prisma db push` thủ công — nhớ chạy trước từ máy local với `DIRECT_URL`.
- Nếu chạy local Docker: đảm bảo container PostgreSQL đang chạy trước khi `prisma db push`.
