## Thay đổi Code

### 1. Lát cắt Tuần/Tháng/Quý/Năm — hiển thị đúng công việc giao với kỳ

**Ảnh hưởng:** Tab **Công việc của tôi** và tab **Quản lý công việc**

**Vấn đề cũ:** Bộ lọc lát cắt chỉ kiểm tra `ngày kết thúc` có nằm trong kỳ không — nên công việc bắt đầu trong tuần nhưng kết thúc tuần sau bị ẩn.

**Hành vi mới:** Hiển thị công việc nếu khoảng `[bắt đầu, kết thúc]` **giao với** kỳ đang chọn, theo quy tắc:
- `plannedStart ≤ ngày cuối kỳ` **VÀ** `plannedEnd ≥ ngày đầu kỳ`
- Công việc chỉ có một trong hai ngày → vẫn hiện (không bị lọc mất)
- Công việc không có ngày nào → luôn hiện

**File:** `src/app/(app)/tasks/tasks-client.tsx`, `src/app/(app)/manage/manage-client.tsx`

---

### 2. Thêm tương tự — Dự án và Loại hình điền đúng

**Ảnh hưởng:** Tab **Công việc của tôi** → chọn 1 việc → "Thêm tương tự"

**Vấn đề cũ:**
- Cột **Dự án** hiện "— Không —" (dropdown không biết chọn nhóm nào)
- Cột **Loại hình (L2)** hiện sai (hiện project code dài thay vì construction type code ngắn)

**Nguyên nhân:**
- Form giao việc dùng `projectGroupId` để hiển thị cột Dự án, nhưng task chỉ lưu `projectId` — thiếu bước tra ngược.
- `task.level2` trong DB đôi khi lưu project code đầy đủ (ví dụ `T.HTI.STHT.NOXH.CT`) thay vì construction type code (`NOXH.CT`).

**Hành vi mới:** Khi prefill từ task có `projectId`, tự động tra ngược:
- `projectGroupId` ← `project.groupId` → dropdown Dự án hiện đúng mã nhóm
- `level2` ← `project.constructionTypeCode` → Loại hình hiện đúng
- `level3` ← `project.name` → Hạng mục hiện đúng

**File:** `src/app/(app)/assign/assign-client.tsx`
