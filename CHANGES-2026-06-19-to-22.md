## 2026-06-22 — Export báo cáo Excel & Bulk edit master data

### 10. Export báo cáo Excel — mở rộng dữ liệu

**Ảnh hưởng:** API `/api/export/reports`

Thêm sheet **"Danh sach cong viec"** với đầy đủ thông tin mỗi công việc:

| Cột mới | Mô tả |
|---|---|
| Mã dự án | `project.group.code` |
| Loại hình | `constructionType.code` |
| Hạng mục | `project.name` hoặc `level3` |
| Bộ môn | `discipline.code` |
| Giai đoạn | `phase.code` |
| Thực hiện | Danh sách người được giao |
| Số người / ghi giờ | Đếm assignee / user đã ghi giờ |
| Ưu tiên, Tình trạng | Label đã dịch |
| Thực tế hoàn thành | `actualEnd` |
| Đo định mức | `measureNorm` |
| Tổng giờ, Số ngày | Tổng hợp từ TimeSheetEntry |
| Định mức | Trung bình giờ/bộ môn-loại hình |

Sắp xếp: theo `workGroup.order` → mã dự án → `constructionType.order` → tên dự án → tên công việc.

Sheet **"Thoi gian theo viec"** và **"Thoi gian viec-nguoi"** bổ sung thêm cột Dự án, Loại hình, Hạng mục, Bộ môn, Giai đoạn, Tình trạng, Thực tế hoàn thành.

Tối ưu: TimeSheetEntry chỉ query theo task đã có trong kết quả (`taskId IN [...]`), tránh query thừa.

**File:** `src/app/api/export/reports/route.ts`

### 11. Bulk edit master data — Khai báo thông tin

**Ảnh hưởng:** Màn **Khai báo thông tin** (tất cả tab)

**Chọn nhiều dòng:**
- Checkbox trên mỗi dòng cho 7 bảng: Nhóm CV, Dự án, Dự án BIM Tools, Công việc, Giai đoạn, Bộ môn, Loại hình công trình.
- Shift+click để chọn cả khoảng.

**Thanh thao tác hàng loạt** hiện khi chọn ≥ 1 dòng:

| Bảng | Trường có thể sửa hàng loạt |
|---|---|
| Nhóm công việc | Mã, Tên, Viết tắt, Thứ tự |
| Dự án / BIM Tools | Dự án, Loại hình, Hạng mục |
| Công việc | Nhóm, Tên |
| Giai đoạn | Mã, Tên, Thứ tự |
| Bộ môn | Mã, Tên |
| Loại hình công trình | Mã, Tên |

**Server actions mới:**
- `batchUpdateSimpleCatalog` — cập nhật nhiều dòng catalog đơn giản.
- Mở rộng `batchUpdateCatalogItems` — đổi nhóm CV cho nhiều `CatalogItem`.

**File:** `admin/catalog/catalog-client.tsx`, `server/actions/catalog.ts`, `server/actions/projects.ts`
